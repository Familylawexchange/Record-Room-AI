🛡️ Source boundary

  • The app treats uploaded files as the complete evidentiary universe.
  • Every generated finding includes a quote and document/page reference when text is extractable.
  • Actor-based results are labeled as fact patterns for attorney review.
  • Local-only storage uses this browser's IndexedDB to simulate private storage; the Node server only receives extracted text when OpenAI analysis is requested.
  • TXT, DOCX, and embedded PDF text extraction run locally; scanned PDFs/images keep a clear local OCR placeholder until a private OCR worker/backend is connected.
  • OpenAI analysis uses OPENAI_API_KEY on the local server and requires citations with document name, page when available, and quoted source text.
